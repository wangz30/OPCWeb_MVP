## 🚨 注意事项

### 安全问题
1. **仅限可信网络使用**: 不要在生产环境中使用`host: '0.0.0.0'`
2. **防火墙设置**: 确保端口3000对局域网开放
3. **文件权限**: 合理设置共享文件夹的读写权限
4. **密码安全**: 考虑创建专用共享账户，避免使用主账户密码

### 开发注意事项
1. **避免同时编辑同一文件**: 可能导致冲突
2. **定期同步代码**: 使用Git拉取最新代码
3. **沟通协调**: 明确分工，避免重复工作

### 性能考虑
1. **热重载可能较慢**: 多个设备同时刷新可能影响性能
2. **网络稳定性**: 确保WiFi信号稳定
3. **浏览器缓存**: 偶尔需要清除浏览器缓存

## 🔍 故障排除

### 无法访问开发服务器？
1. 检查防火墙设置
   ```bash
   # macOS
   sudo /usr/libexec/ApplicationFirewall/socketfilterfw --add /Applications/Google\ Chrome.app
   ```

2. 检查端口是否被占用
   ```bash
   lsof -i :3000
   ```

3. 重启开发服务器
   ```bash
   pkill -f "vite"
   npm run dev
   ```

### 文件无法保存？
1. 检查文件权限
   ```bash
   chmod -R 755 /Users/iris/CodeBuddy/20260313105540/opc-cloud-community
   ```

2. 使用正确的编辑器权限
