# OPC Dashboard Demo

OPC（Open Platform for Communication）管理后台系统演示项目。

## 📁 项目结构

```
Dashboard_Demo/
├── admin.html          # MaaS 管理后台界面
└── opc-backend.html    # OPC 用户后台界面（深国创）
```

## 🚀 功能模块

### 管理后台（admin.html）
- **模型管理** - AI 模型上架、价格配置、调用统计
- **用户管理** - 用户信息、使用数据、状态管理
- **数据看板** - 实时调用监控、趋势分析、模型分布
- **财务管理** - 交易记录、收入统计、结算管理
- **创客空间** - 入驻申请审核
- **创业活动** - 活动报名管理

### 用户后台（opc-backend.html）
- 资源对接
- 补贴申请
- 需求匹配
- 进度跟踪
- 消息提醒

## 🛠️ 技术栈

- 纯 HTML/CSS/JavaScript 实现
- Font Awesome 图标库
- Google Fonts 字体
- 响应式设计

## 📖 使用方法

直接在浏览器中打开对应的 HTML 文件：

```bash
# 打开管理后台
open admin.html

# 打开用户后台
open opc-backend.html
```
