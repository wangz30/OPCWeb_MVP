## 📖 项目简介

**云上 OPC 社区**是一个 AI 驱动的一人公司创业平台，由深圳国家高技术产业创新中心、粤港澳大湾区大数据中心支持建设。平台整合政策资源、提供 AI 工具服务、连接创业生态，助力个体创业者把握 AI 时代机遇。

### 核心定位
- 🎯 **AI 创业赋能**：提供大模型训练、智能体开发、数据 API 等一站式 AI 服务
- 📊 **政策对接**：汇聚粤港澳大湾区 AI 专项政策与普惠政策，提供智能券/训力券申请指引
- 🤝 **生态连接**：创享会、创客空间、资源对接，构建一人公司创业生态圈

## ✨ 功能特性

### 🚀 核心服务
| 服务模块 | 功能描述 |
|---------|---------|
| **云资源服务** | 低成本算力租赁、GPU 资源调度、训力券申请指引 |
| **AI 模型服务** | 大模型 API 接入、智能体开发工具、模型微调服务 |
| **数据 API 服务** | 行业数据集、语料库资源、数据标注服务 |
| **创客空间入驻** | 低成本工位申请、OPC 社区"拎脑入驻"、落户补贴指引 |

### 📰 资讯与政策
- **政策速递**：广东省/深圳市/广州市 AI 专项政策实时更新
- **科普好文**：AI 创业趋势、商业案例、技术解读
- **活动动态**：云上 OPC 创享会、创业沙龙、资源对接会

## 🛠️ 技术栈

### 前端
- **框架**: React 19 + TypeScript
- **构建工具**: Vite 8
- **路由**: React Router DOM 7
- **UI 组件**: Lucide React 图标库
- **样式**: Tailwind CSS 3 + PostCSS
- **工具库**: class-variance-authority, clsx, tailwind-merge

### 部署
- **平台**: Vercel
- **域名**: opccomunity.vercel.app
