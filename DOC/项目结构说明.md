```

│   ├── 首页
│   ├── 产品中心
│   ├── 解决方案中心
│   ├── 应用集市
│   ├── 培训活动
```

## 🗺️ 完整页面层级结构

### 一级导航（主菜单）

```
导航栏 (6 个主菜单项)
├── 首页 (/)
├── 产品 (/products)
├── 解决方案 (/solutions)
├── 应用集市 (/marketplace)
├── 培训活动 (/activities)
└── 关于我们 (/about)
```

---

### 二级导航（子菜单）

#### 1. **产品** `/products`
```
产品 (一级)
├── AI 模型 (/models)
├── 数据 API (/data-apis)
└── 云资源 (/cloud-resources)
```

**页面内容**:
- **AI 模型**: 展示各类 AI 模型（语言、图像、代码、视频等）
- **数据 API**: 提供各类数据接口服务（金融、气象、地理、电商等）
- **云资源**: 云计算、存储、数据库、网络等资源

---

#### 2. **解决方案** `/solutions`
```
解决方案 (一级)
├── 信息与政策赋能 (/policies)
├── AI 技术与云资源 (/cloud-resources)
└── 全维度资源与能力升级 (/resources)
```

**页面内容**:
- **信息与政策**: 政策资讯、融资政策、人才政策
- **AI 技术与云资源**: AI 模型 + 云资源组合方案
- **全维度资源**: 课程、需求对接、资源对接等

---

#### 3. **培训活动** `/activities`
```
培训活动 (一级)
└── 活动回顾 (/events/review)
```

**页面内容**:
- **活动列表**: 即将举办的活动、在线课程、训练营
- **活动回顾**: 历史活动的图文回顾

---

### 独立页面（无二级菜单）

```
├── 首页 (/)
├── 应用集市 (/marketplace)
└── 关于我们 (/about)
```

## 📱 响应式断点

```css
/* 移动端 */
@media (max-width: 768px) {
  - 导航栏折叠为汉堡菜单
  - 网格布局变为单列
  - 字体大小调整
}

/* 平板端 */
@media (min-width: 768px) and (max-width: 1024px) {
  - 导航栏部分展开
  - 网格布局为 2 列
}

/* 桌面端 */
@media (min-width: 1024px) {
  - 完整导航栏
  - 网格布局为 3-4 列
  - 显示完整功能
}
```
