# 团队协作身份验证指南

## 📋 问题解答
**问题**: 团队成员输入地址后需要填用户名和密码，是我的吗？

**答案**: **是的，需要你的Mac电脑的用户名和密码**。

## 🔑 你需要提供的信息

### 1. **你的Mac用户名**
- 打开终端，输入: `whoami`
- 或者查看: 系统设置 → 用户与群组

### 2. **你的Mac登录密码**

## 🛡️ 安全替代方案（推荐）

### 方案A：创建专用共享账户（最安全）
1. 打开"系统设置" → "用户与群组"
2. 点击"添加账户" (+)
3. 选择"仅限共享"
4. 创建账户: `shared-dev` (示例)
5. 设置简单密码: `dev123` (示例)
6. 在"系统设置" → "共享" → "文件共享"中，只给这个账户访问项目文件夹的权限
7. 分享这个账户的凭据给团队成员

### 方案B：无密码方案（推荐）

#### 1. **实时预览（无需编辑权限）**
- **分享这个URL给团队成员**:
  ```
  http://192.168.190.57:3000
  ```
- **团队成员可以**: 查看实时效果，无法编辑代码

#### 2. **代码编辑（无需密码）**
- **使用VS Code Live Share**:
  1. 在VS Code安装"Live Share"扩展
  2. 点击左下角"Live Share"按钮
  3. 分享链接给团队成员
  4. 团队成员可以实时编辑，无需密码

#### 3. **Git协作（无需密码）**
1. 创建GitHub/GitLab仓库
2. 推送代码到远程仓库
3. 团队成员克隆代码到本地编辑
4. 通过Pull Request协作

## 📞 紧急情况处理

### 如果忘记密码
1. **重置Mac密码**:
   - 重启Mac，按住Command+R进入恢复模式
   - 选择"实用工具" → "终端"
   - 输入: `resetpassword`

### 如果不想重置密码
1. **只使用预览功能**: 分享 `http://192.168.190.57:3000`
2. **使用在线工具**: GitHub Codespaces, CodeSandbox等
